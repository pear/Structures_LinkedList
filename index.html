Structures_LinkedList package for PHP

Hey, welcome to the Structures_LinkedList package for PHP. This package provides both a singly-linked list (Structures_LinkedList_Single) and a doubly-linked list (Structures_LinkedList_Double).

Requirements

This package requires PHP 5, and was primarily developed with PHP 5.1.4. I have also successfully tested 5.2.0RC1 through RC4.

Installation

  1. Download the Structures_LinkedList package.
  2. Forcefully install the package with the PEAR installer: 
    # pear install --force Structures_LinkedList-0.0.#.tgz
